Personal portfolio built from scratch — minimal, clean, and intentionally over-engineered in zero places. No templates. Just code, design decisions, and a lot of self-teaching.

## About

I'm Francis — a self-taught developer working my way through the full stack one project at a time. This portfolio is both a showcase and a learning artifact: the code here reflects how I think, what I've figured out, and where I'm heading.

Currently available for work.

## Stack

| Layer | Tool |
|---|---|
| Framework | React 19 |
| Bundler | Vite |
| Styling | Tailwind CSS v4 (via `@tailwindcss/vite`) |
| Animations | Framer Motion |
| Language | JavaScript (no TypeScript) |

## Getting Started

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Production build
npm run build

# Preview production build
npm run preview
```

## Project Structure

```
src/
├── assets/         # Images, icons, fonts
├── components/     # Reusable UI components
│   ├── Hero/
│   ├── MyWork/
│   └── UI/
├── pages/          # Page compositions
├── styles/         # Global styles (Tailwind import)
├── App.jsx         # Root component
└── main.jsx        # Entry point
```
